# Contract Mirror UI v5.3.1 Judge Final Card Fix

## 목적

`/judge` 마지막 검증 카드 단계에서 중앙 검증 카드가 좁은 슬롯 안에 들어가 세로로 찌그러지는 문제를 수정한다.

## 수정 내용

- Judge 라우트에서 모바일 쉘을 완전히 제거하도록 CSS 우선순위 강화
- 중앙 컬럼 최소 폭 확보
- 마지막 검증 카드 단계에서 `body.judge-final-step` 클래스 적용
- 검증 카드 그리드 2단 레이아웃 고정
- 오른쪽 Trust Summary 폭 고정
- Raw Trust Log는 보조 정보로 유지
- 긴 URL/hash 값 줄바꿈 처리

## 정상 기준

- `/judge` 마지막 검증 카드 상태에서 “계약 설명 검증 카드 생성 완료”가 세로로 찢기지 않는다.
- 중앙 검증 카드가 대시보드의 클라이맥스처럼 넓게 보인다.
- 오른쪽 Trust Summary가 중앙 영역을 밀어내지 않는다.
- 가로 스크롤이 생기지 않는다.
