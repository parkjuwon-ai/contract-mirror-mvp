# UI v6.12 Simplified Core Flow

## 핵심 변경

- `/report`, `/verify`, `/addendum`에서 모바일 폰 프레임 문법을 제거하고 PC/태블릿용 워크스페이스로 분리
- 사용자 모바일 화면은 한 화면 한 행동 원칙으로 압축
- 첫 화면/계약서 등록/대기/녹취 화면의 설명 문구를 줄이고 핵심 행동을 유지
- `/report`는 “계약 전 확인할 위험 3건” 중심의 문서형 리포트로 정리
- 질문은 카드별 개별 복사만 유지
- 검증 카드에서는 `VALID` 대신 “기록 일치”로 표현해 법적 오해 가능성을 줄임
- 기술 정보는 사용자 화면에서 접힘 상태 유지, judge 화면에서만 자세히 확인

## 백엔드 연결 안전성

- 기존 FastAPI 라우트 유지: `/`, `/report/{id}`, `/addendum/{id}`, `/verify/{id}`, `/judge`, `/participant/*`
- 기존 `data-action` 기반 클릭 흐름 유지
- API/백엔드 데이터 스키마를 새로 요구하지 않음
- 향후 실제 업로드, STT, AI 분석, QR 생성, Chain 연동 시 현재 DOM의 표시 영역만 실제 데이터로 치환 가능
