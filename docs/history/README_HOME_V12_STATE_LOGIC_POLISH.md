# Contract Mirror Home V12 State Logic Polish

반영 내용:

- 초대 코드 화면 상태 로직 정리
  - 상대 입장 전에는 `상대 입장 대기 중`으로 표시
  - 데모 진행용 `설명자 입장 완료 처리` 버튼 추가
  - 준비 완료 후에만 `다음: 녹취 동의 확인하기` 활성화
- 홈 CTA 첫 프레임 노출을 위해 홈 화면 높이/여백 압축
- 추가 자료 화면을 선택 보강 단계로 재정의
  - `검증 카드를 더 강하게 남길까요?`
  - `선택 사항입니다. 자료가 없어도 검증 카드를 만들 수 있습니다.`
- `확인 카드` 용어를 `검증 카드`로 통일
- 검증 카드 화면에 기록 정보 명확화
  - 검증 기록 ID, 생성 시각, 위험 항목 수, QR 검증 페이지
- 결과 리포트 오른쪽 다음 행동 패널 단순화
