# Contract Mirror UI v6.9 Protection Flow Polish

## 목적
위험 발견 이후 사용자가 현장에서 바로 질문하고, 확약서/특약을 요청하고, 검증 기록으로 남기는 행동 중심 UX를 강화한다.

## 변경 사항
- 첫 화면 카피를 실제 위험 상황 중심으로 변경
- 상단 시작 상태 제거 및 하단 CTA 집중
- 위험 예시를 흐름형 비교 UI로 변경
- 한 화면 Primary 버튼 하나 원칙 강화
- 동의/추가자료/검증카드 문구 압축
- 녹취 화면에 양측 본인확인 상태와 계약서 연결 상태 고정
- 결과 화면 상단에 계약 전 확인 질문 배치
- 결과 버튼을 검증 카드 생성 / 질문 복사 / 추가자료 등록 중심으로 정리
- 검증 카드 화면은 사용자 쓸모를 먼저 보여주고 기술 정보는 접힘 처리
- DID 지갑 연동은 후속 확장 카드로만 표시
