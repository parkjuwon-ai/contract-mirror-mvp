# Contract Mirror UI v6.4 Step 3 Responsive Report Polish

## 목표

v6.4는 v6 Responsive MVP의 3단계로, 결과 리포트와 검증 카드가 사후 검토 화면처럼 보이도록 정리한 버전이다.

## 반영 사항

- 사용자 플로우의 폰 바깥 단계바 제거
- 모바일 목업 크기와 가독성 개선
- 대기 화면을 계약자/설명자 관제 카드 중심으로 강화
- 결과 화면에 다음 조치 영역 추가
- `검증 카드 생성하기`를 1차 CTA로 강화
- `검증 리포트 저장하기`, `재설명 요청하기`를 보조 CTA로 낮춤
- 결과/검증 단계에서는 스마트폰 목업이 아니라 넓은 사후 검토 리포트로 표시
- 모바일 실제 접속에서는 폰 프레임 제거
- 모바일 프레임 내부 스크롤바 숨김 유지

## UX 원칙

- 현장 입력은 모바일 웹
- 결과 검토는 반응형 리포트
- 심사위원 검증은 Trust Audit Dashboard
