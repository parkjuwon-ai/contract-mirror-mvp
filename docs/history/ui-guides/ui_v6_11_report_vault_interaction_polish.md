# UI v6.11 Report & Vault Interaction Polish

## 목적

v6.11의 목적은 계약미러가 단순히 리포트를 보여주는 목업이 아니라, 사용자가 실제로 질문을 복사하고 검증 카드를 보관하는 앱처럼 느껴지게 만드는 것입니다.

## 반영 내용

1. `/report` 확장 화면 정리
   - 리포트 전용 헤더 추가
   - 계약 ID와 기준 계약서 표시
   - 질문 카드, 불일치 근거, 다음 행동 패널을 문서형 워크스페이스로 분리

2. 질문 카드별 복사
   - 큰 `질문 복사하기` 버튼 제거
   - 질문마다 작은 `📋 복사` 버튼 추가
   - 복사 시 토스트 표시

3. 검증 카드 보관 플로우
   - `검증 카드 저장하기` → `검증 카드 보관하기`
   - 보관 중 / 보관 완료 바텀시트 추가
   - 계약미러 보안 보관함 개념 도입

4. 검증 카드 화면 정리
   - 저장 위치 표시
   - 기술 검증 정보 접힘 유지
   - 공유용 링크 복사 액션 추가

5. Judge 대시보드 연동
   - Trust Summary 및 Raw Log에 보관 상태 반영
