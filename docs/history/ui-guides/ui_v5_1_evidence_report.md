# Contract Mirror UI v5.1 Evidence Report

v5.1은 v5의 사용자 흐름을 유지하면서 결과·해시·QR 검증 화면을 증거 리포트 중심으로 강화한 버전입니다.

## 핵심 변경

- 첫 화면: 체크리스트형 안내 제거, 행동 버튼 중심으로 재배치
- 용어: 계약 설명방 → 계약 설명 세션
- 동의 버튼: 확인했고 참여할게요 → 본인확인 및 녹취에 동의합니다
- 계약서 기록: 변경 확인용 기록 → 계약서 원본 해시 생성
- 대기 화면: 텍스트 나열 대신 상태 패널과 대기 표시 추가
- 결과 화면: 위험도, 녹취 위치, 계약서 조항, 좌우 비교, AI 판단 이유 추가
- 검증 화면: 세션 ID, 계약서/녹취/리포트 해시, QR 검증 카드 강화
- 심사위원 모드: Trust Log에 리포트/검증 카드 상태 추가

## 주요 확인 URL

- `/` 사용자 첫 화면
- `/judge` 심사위원 기술 검증 모드
- `/participant/contractor` 계약자 본인확인·동의 화면
- `/participant/explainer` 설명자 본인확인·동의 화면

## 확인 기준

- 사용자 화면에서 심사위원 모드 버튼이 보이지 않아야 합니다.
- 결과 화면에서 불일치 후보 3개, 위험도, 녹취 위치, 계약 조항, 판단 이유가 보여야 합니다.
- 검증 화면에서 계약미러 검증 카드와 QR 박스가 크게 보여야 합니다.
- 심사위원 화면에서 Trust Log, mock/simulated 상태, 검증 카드 상태가 보여야 합니다.
