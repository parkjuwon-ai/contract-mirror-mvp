# Contract Mirror Home V9 Flow Logic Patch

## 반영 내용

- 계약서 등록/확인 단계에서 결과 이후용 `확약서/추가 자료 등록`으로 빠지는 버튼 제거
- 초대 코드 진입 플로우를 계약자 플로우와 분리
- 초대 코드 입력 → 녹취 동의 → 동의 완료 → 녹취 참여 → 참여 완료 흐름 추가
- 초대받은 사람은 AI 분석 시작 주체가 아니라 녹취 참여자로 표현
- 결과 이후 `확약서/추가 문서 등록`은 리포트 화면에서만 접근

## 핵심 의도

계약자 플로우와 초대받은 사람 플로우를 분리해서, 녹취 전 단계에서 확약서 화면으로 점프하거나 초대 코드 참여 후 화면이 끊기는 문제를 정리했습니다.
